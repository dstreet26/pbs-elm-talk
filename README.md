# pbs-elm-talk
Slides for PBS Tech Talks for Elm


# Quickstart

```
git clone https://github.com/dstreet26/pbs-elm-talk
cd pbs-elm-talk
open Part\ 1\ Intro\ to\ Elm.html
```

To see slide notes, press "P" to go into Presenter mode.


## REPL follow-along

There are some files called `talk_repl_yminutes_[1,2,3].elm` that are paste-able into `elm repl`.

The content comes from https://learnxinyminutes.com/docs/elm/.

It is split into 3 parts to make it easier on the repl.
